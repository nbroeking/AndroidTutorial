Getting started

Download Android Studio Installer

Navigate to http://developer.android.com/intl/ja/sdk/index.html and select "Download". The download should be ~ 1 gig.

Install Java JDK

Android studio requires the Java JDK. If you do not have java installed, it can be downloaded from: http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html. If you already have java, skip this step.

Run the installer

Running the installer and installing all the defaults will add ~4 gigs, but all components are suggested as they include the SDK, Virtual Device, etc.
