Introduction

Android is a powerful platform developed by google for their mobile phones. Over 18,000+ phones run android and take up about 80% of market share. Google provides a great toolchain to develop native applications for their android platform. This android sdk provides developers all the tools that we need to develop powerfull applications.


What is the Android SDK?

The android SDK is the software development kit provided by google that provides all the tools to create android apps. Apps are written in java and run on top of the darwin platform. Darwin is googles app manager that runs with the os to provide inter application communication. We will talk more about this later. The sdk provides everything java related but it doesnt stop there. It provieds debuging tools such as adb that allows developers to shell into the phone, install apps, and pretty much anything else you wanted to do.

Why develop for android?

Android is a powerful platform and as we mentioned before, android takes up 80% of the mobile market share. You can reach a huge number of customers just by developing your app for android. Another plus is the fact that you are writting native applications. The application will compile and run directly on the device allowing you access to powerfull processing power as well as direct graphics capabilities. Finally, Its easy!!! Android is very easy to develop. As we are going to show you anybody can learn it.
