Introduction

Android is a powerful platform developed by google for their mobile phones. Over 18,000+ phones run android [source] and take up about 80% of market share.[source] Google provides a great toolchain to develop native applications for their android platform. This android sdk provides developers all the tools that we need to develop powerfull applications.

What is the Android SDK?

The android SDK is the software development kit by google that provides all the tools to create android apps. Apps are written in java and run on top of the darwin platform. Darwin is googles app manager that runs with the OS to provide inter application communication. We will talk more about this later. The SDK provides everything java related but it doesn't stop there. It provides debugging tools such as ADB that allows developers to shell into the phone, install apps, and pretty much anything else they want to do.

Why develop for android?

Android is a powerful platform and, as we mentioned before, android takes up 80% of the mobile market share. You can reach a huge number of customers just by developing your app for android. Another plus is the fact that you are writing native applications. The application will compile and run directly on the device allowing you access to powerful processing power as well as direct graphics capabilities. Finally, It's easy!!! Android is very easy to develop for. As we are going to show you, anybody can learn it.

What should I know before looking at this page?

A familiarity with java is extremely helpful but at a minimum you should at least know one programming language as well as basic software engineering fundamentals.
